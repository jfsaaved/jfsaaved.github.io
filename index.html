Portfolio

A list of applications I have made.

Space Hunter

A 2D "life in space" simulator role-playing game. This is my entry for the Jan. 2016 libGDX Jam. Users will play as a 'space hunter' and travel through the galaxy with a space ship.

Adventures of Kobe

A 2D Endless Runner game developed in Java and libGDX with its own online leader board. Player earn points by surviving or destroying incoming objects.

4x4 Tiles

A pattern memory game developed in Java and libGDX. Player earn points by imitating a pattern shown beforehand.

Discord Movie Bot

A movie bot for Discord, created with Java. Uses OMDBapi to query movies. Great for movie nights where a group is having trouble picking a movie.
